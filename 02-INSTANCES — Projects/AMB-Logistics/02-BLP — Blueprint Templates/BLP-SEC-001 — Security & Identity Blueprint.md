# 7. Protected Identity Runtime Rule
Las identidades administrativas `admin.jan@amblogistics.be`, `admin.bram@amblogistics.be`, y `breakglass@amblogistics.be` quedan protegidas contra mutacion automatizada. El runtime debe devolver `SKIPPED_PROTECTED` o `BLOCKED` ante cualquier intento de modificacion de estas cuentas. Los ObjectId se documentan como `UNKNOWN` hasta validacion real del tenant.
